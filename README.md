# Portafolio de Mampostería - Jairo Cibrian

Sitio web profesional para mostrar trabajos de mampostería especializada: enchapado de ladrillo antiguo, mampostería heritage y dry stack artesanal.

## Características

- Diseño responsivo con Tailwind CSS
- Comparador interactivo antes/después (slider)
- Galería completa para cada tipo de trabajo
- Soporte bilingüe (inglés/español)
- Modal de contacto directo (llamada, SMS, email)
- Optimizado para móviles y desktop

## Cómo usar

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/jairo-masonry-portfolio.git
   ```

2. Abre `index.html` en cualquier navegador moderno.

3. Para personalizar:
   - Reemplaza las imágenes en la raíz con tus propias fotos.
   - Modifica los textos en `index.html` o en el objeto `translations` del JavaScript.
   - Actualiza los datos de contacto en la sección final.

## Despliegue en GitHub Pages

1. Sube el proyecto a un repositorio GitHub.
2. Ve a **Settings > Pages**.
3. En **Source**, selecciona la rama `main` (o `master`) y carpeta `/root`.
4. Guarda y espera unos minutos. Tu sitio estará disponible en `https://tu-usuario.github.io/nombre-repo/`.

## Tecnologías utilizadas

- HTML5, CSS3, JavaScript ES6
- [Tailwind CSS](https://tailwindcss.com/) para estilos
- Google Fonts (Inter, JetBrains Mono)
- SVG para patrón de fondo
- CSS Grid & Flexbox
